Zahlenmaschine

The "Zahlenmaschine" is a simple computer, programmed with an assembly-like language.

Accumulator: 0

Instruction pointer: 0

Status: false

Data r1: 0

Data r2: 0

User guide

Numbers are arbitrary integers. Storage may be one of: acc (Accumulator), sta (Status Register), isp (Instruction Pointer), r1 (data register 1), r2 (data register 2), nul (Immutable Zero) . Value may be anything. Comments may be added with the ';' character. All content in the line behind this character will be ignored.
Operation arg1 arg2 Explanation
acc Number - Add Number (may be negative) to accumulator.
Jumps
jmp Number/Label - Jump to Number/Label
jre Number - Jump Number instructions (relative).
jtr Number/Label - Absolute jump to Number/Label only when status is true.
Comparisons
grt Storage Number If value in Storage is greater than Number, set status to true
geq Storage Number If value in Storage is greater or equal to Number, set status to true
equ Storage Number If value in Storage is equal to Number, set status to true
leq Storage Number If value in Storage is less or equal to Number, set status to true
les Storage Number If value in Storage is less than Number, set status to true
Working with the data register
add Storage Number Add value in Storage to Number, write result into data register r1
adr Storage1 Storage2 Add values in both Storages, write result into data register r1
sub Storage Number Subtract Number from value in Storage, write result into data register r1
mul Storage Number Multiply Number with value in Storage, write result into data register r1
neg Storage - Negate value in Storage, write result into data register r1
mov Storage1 Storage2 Copy value from Storage1 to Storage2
swp Storage1 Storage2 Copy value from Storage1 to Storage2 and value from Storage2 to Storage1
Control operations
nop - - Do nothing
rst - - Reset accumulator and data register to 0, reset status to false
end Value - Stop machine gracefully with return value

Zahlenmaschine

The "Zahlenmaschine" is a simple computer, programmed with an assembly-like language.

User guide

Numbers are arbitrary integers.

Storage

Storage describes places in the machine's memory. There is no memory layout but there are some registers.
  • acc - Accumulator, may be used as a general purpose register
  • sta - Status register, stores results of comparisons
  • isp - Instruction pointer, determines the next instruction
  • r1 - Data register 1, stores results of data operations
  • r2 - Data register 2, may be used to store data
  • Read only Storage
  • nul - Null, always 0
  • rnd - Random, reading from this register will yield a random number. Ranges may be set with the rnr instruction
  • stc - Stack count

Comments may be added with the ';' character. All content in the line behind this character will be ignored. Labels can be added by writing a string and ending it with the ':' character.
